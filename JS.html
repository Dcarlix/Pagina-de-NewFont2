Comienza el curso

JavaScript

El lenguaje de JavaScript es un lenguaje de programación muy utilizado para el desarrollo de paginas web como la pagina que estas viendo. Pero seamos mas especificos, te introduciremos a que es este lenguaje que, que hace, donde se usa, y sobre todo porque deberias contratar un curso con nosotros para que puedas aprender a usar este lenguaje de programacion en tus paginas web.




¿Qué es JavaScript?


JavaScript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás. Y aunque quiza no todo, tambien puede hacer ciertas cosas posibles en otros lenguajes pero con menos lineas de codigo.




¿Qué hace JavaScript?


Lo que hace JavaScript es agregarle funcionalidades a tu pagina web, es decir, le agrega a tu pagina efectos y acciones que son muy dificiles de hacer solo con CSS o practicamente imposibles de hacer sin la ayuda de JS, por ejemlo la cabezera de la pagina de NewFont que tu estas viendo tiene un codigo de JS que hace que la cabezera cuando esta encima de la imagen de incio se vuelva transparente, pero en el momento en el que bajas y comienzas con el contenido de la pagina este se vuelva gris. El codigo de lo tienes a tu derecha.




¿Por que es necesario JavaScript?


Dentro de la programacion web tenemos tres lenguajes importantes: HTML5, CSS3 y JavaScript, y para hacerte comprender la importancia de los tres, te lo compararemos al cuerpo humano; HTML5 es como el esqueleto de tu pagina, porque este lenguaje es donde definimos por bloques y estructuras de diferentes niveles y tamaños que le dan la base a tu pagina, CSS3 es como la apariencia de una persona, ya que con CSS es como le damos el color, tamaño, sombra, espacio, efectos y demás cosas a la pagina para que tenga un mejor estilo, y por ultimo JS es como el sistema nervioso y el cerebro, porque este lenguaje es el que le da las funcionalidades (o la mayoría de ellas) a tu pagina web, es la que puede guardar datos, procesarlos, y dar una respuesta a esos datos.

Un poco de JavaScript


El Lenguaje de JS es un poco parecido a lenguajes como C#, C++ O Phyton; Por lo que comparte la cualidad de ser case-sensitive, lo que significa que si creamos una variable (cosa que se puede hacer en JS) que se llame Vaso, esta variable sera diferente de vaso porque tienen una letra diferente apesar de ser iguales.


Tambien comparte la sintaxis de muchas estructuras que quiza ya conoscas, como:

  • Declaración for
  • Declaración do...while
  • Declaración while
  • Condicional if...else
  • Condicional switch
  • Expresión trhow
  • Declaración try...catch


Pero JS tambien contiene una nuevas formas y nuevas sintaxis para escribir algunas instrucciones que ya conocias o para escribir nuevas instrucciones que no conocias o no sabias que se podian hacer en este lenguaje, como los tipos de variables que existen en JS, los Arrays, trabajar con programación orientada a objetos, trabajar con programacion de datos Meta, seleccionar elementos de HTML5 y de CSS3 por diferentes metodos, agregarle nuevas propiedades a estos, y muchas otras cosas más.


Por eso si tu deseas conocer más sobre este lenguaje y otros mas, ve a la pagina de cursos y consigue uno con unos de nuestros docentes de NewFont, para que te puedan ayudara a aprender de este, y otros lenguaes de programación.
