¿Qué es HTML?

HTML significa "lenguaje de marcado de hipertexto" y es el lenguaje de marcado estándar utilizado para crear páginas web. Fue creado para crear contenido de páginas web utilizando etiquetas o etiquetas que definen la estructura y el significado de los elementos de la página.

Utilidad

HTML es esencial para la creación y visualización de páginas web en la World Wide Web. Su utilidad radica en su capacidad para estructurar y organizar el contenido de una página web de manera que los navegadores web puedan interpretarlo y mostrarlo de manera coherente.

Etiquetas

Te mencionare algunas etiquetas usadas comunmente en HTML.

Estructura de documento

!DOCTYPE
Define la versión de HTML utilizada en el documento.
HTML
Envuelve todo el contenido de la página.
HEAD
Contiene información sobre el documento, como el título de la página, metadatos y enlaces a archivos CSS o JavaScript.
TITLE
Define el título de la página que se muestra en la barra de título del navegador.
meta
Se utiliza para incluir metadatos, como el conjunto de caracteres, descripciones y palabras clave.

Contenido del documento

BODY
Contiene todo el contenido visible de la página, como texto, imágenes, enlaces y otros elementos.
h1, h2, h3, etc..:
Se utilizan para crear encabezados con diferentes niveles de importancia.
P
Se utiliza para crear párrafos de texto.
SPAN
Se utiliza para aplicar estilos o manipular partes específicas de texto dentro de un párrafo o bloque de texto.
div
Se utiliza para agrupar y estructurar elementos en bloques, muy usado en esta pagina creada, dato, este texto esta dentro de un div class.

Semántica y Organización

Header
Define la cabecera de una sección o página.
footer
Define el pie de página de una sección o página.
nav
Define la navegación de una página.
section
Define una sección dentro de un documento, lo he utilizado igual mucho en esta pagina, una seccion para cada cosa y tenerlo mas ordenado.
aside
Define contenido relacionado o secundario en un documento, la mayoria de veces lo hemos ocupado a la derecha como para poner anuncios o cualquier otra cosa relevante al tema.

Formularios y Entrada de Datos

input
Define campos de entrada en un formulario, como cajas de texto, botones de radio y casillas de verificación.
textarea
Crea un área de texto para la entrada de múltiples líneas.
label
Etiqueta descriptiva para elementos de formulario.
form
Se utiliza para crear formularios interactivos.
button
Para crear botones en formulario.
